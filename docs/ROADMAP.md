# Roadmap

This roadmap lists future upgrades only. These features are not part of the current packaged starter pack.

## Phase 11: Blog CRUD And Content Management

- Admin blog list
- Create/edit/archive posts
- Public blog detail pages
- SEO fields for posts

## Phase 12: Customer Accounts

- Customer login/signup
- Order history
- Saved contact details
- Account profile page

## Phase 13: SMS Notifications

- SMS provider selection
- Order confirmation SMS
- Status update SMS
- Admin SMS notification option

## Phase 14: Real Payment Gateway Integration

- bKash gateway integration
- Nagad gateway integration
- Webhook verification
- Paid status automation
- Payment failure/cancel handling

## Phase 15: Advanced Delivery Tracking

- Courier tracking fields
- Public delivery timeline
- Admin courier status updates
- Optional courier API integration

## Phase 16: Advanced Email Branding And Logs

- Branded HTML templates
- Email preview tools
- Email delivery logs
- Retry visibility

## Phase 17: Bulk Product Import/Export

- CSV import
- CSV export
- Validation report before import
- Bulk archive/publish

## Phase 18: Media Library

- Image browser
- Image reuse across products
- Delete unused uploads
- Folder or tag organization

## Phase 19: Advanced Variant Combinations

- Variant combination matrix
- Per-combination SKU
- Per-combination stock
- Per-combination price
- Out-of-stock option disabling

## Phase 20: Multi-Template Packaging

- Separate demo brands
- Starter presets for fashion, cosmetics, food, seasonal fruit, and retail
- Shared implementation checklist
- Sales landing/demo bundle
