# Portfolio Presentation Guide

Use this guide to present TinyTots BD as a portfolio project, sales demo, or client starter-pack example.

## Short Project Summary

TinyTots BD is a production-ready e-commerce starter pack for Bangladesh small businesses. It combines a mobile-friendly storefront, Supabase-backed product and order management, admin product CRUD, image uploads, inventory, variants, manual bKash/Nagad payment handling, customer order tracking, and deployment documentation.

## Problem It Solves

Many small businesses sell through Facebook, Instagram, WhatsApp, or phone calls but struggle with scattered product information, manual order tracking, and no professional web presence. This starter pack gives them a structured storefront and admin workflow without building every store from scratch.

## Target Client Types

- Baby products stores
- Fashion and clothing shops
- Cosmetics and skincare sellers
- Mango and seasonal product sellers
- Small retail shops
- Facebook and Instagram sellers
- Simple food/product sellers

## Key Business Benefits

- Professional storefront instead of only social media posts
- Customers can browse products anytime
- Store owner can manage products without developer help
- Orders are saved in Supabase
- Owner can verify manual payments
- Customers can track order status
- Inventory can prevent overselling
- Template can be rebranded for new businesses quickly

## Key Technical Features

- Next.js App Router
- TypeScript
- Supabase Auth, Postgres, and Storage
- Protected admin routes
- Server-side trusted checkout calculations
- Stock-aware order creation
- Customer order tracking API
- Optional Resend email hooks
- SEO basics, sitemap, robots, and structured data

## Customer Journey To Show

1. Customer opens home page.
2. Customer browses `/shop`.
3. Customer opens a product detail page.
4. Customer selects a variant and adds to cart.
5. Customer checks out with COD, bKash, or Nagad.
6. Customer receives an order number.
7. Customer tracks the order using order number and phone.
8. Admin updates order/payment status.

## Suggested Portfolio Description

TinyTots BD is a reusable e-commerce starter pack for Bangladesh small businesses. Built with Next.js, Supabase, and Tailwind CSS, it includes product CRUD, image upload, inventory, variants, COD/bKash/Nagad manual order flow, admin order management, customer order tracking, SEO basics, and deployment documentation. The project is designed as a sellable template that can be rebranded for baby products, fashion, cosmetics, seasonal products, and small retail clients.
